# RimpsychicMod

A Rimworld mod that adds weapons and psycasts focused on psychic powers from popular media.

## Overview

Experience the power of psycasts with weapons and abilities inspired by popular media. Expand your colony's psychic arsenal with unique gameplay mechanics and balanced content.

## Features

- **Psychic Weapons** - New combat tools harnessing psychic energy
- **Psycasts** - Custom psychic abilities for your pawns, based on psychic powers from fiction
- **Balanced Gameplay** - Carefully designed for vanilla and modded playthroughs, without having overhead

## Dependencies

- **Vanilla Psycasts Expanded** (required)
- **Combat Extended** (optional, compatible)

## Installation (for nerds)

1. Install dependencies first, from Steam Workshop or Github
2. Download and place in your RimWorld local mods folder
3. Load after CE and VPE in mod list
